# Grade Calculator (Java)

This project is a simple Java program that allows users to enter multiple grades, calculate the average, and determine the corresponding letter grade. It was developed as part of a programming assignment to demonstrate problem-solving, algorithm design, and testing.

---

## 📌 Features
- Accepts user input for multiple grades
- Calculates the average grade
- Assigns a letter grade (A, B, C, D, F)
- Displays results clearly in the console

---

## 🧠 How It Works
1. The user enters how many grades they want to input
2. The program collects each grade
3. It calculates the total and average
4. Based on the average, it assigns a letter grade:
   - 90+ = A  
   - 80–89 = B  
   - 70–79 = C  
   - 60–69 = D  
   - Below 60 = F  

---

## ▶️ How to Run the Program

### Using IntelliJ IDEA
1. Open the project in IntelliJ IDEA
2. Navigate to the `GradeCalculator.java` file
3. Click the **Run** button
4. Enter grades in the console when prompted
